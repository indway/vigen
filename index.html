Frequently Asked Questions

Quick details on VIGEN Architecture expectations.

Apakah sistem ini wajib langganan bulanan?

Is this a monthly subscription SaaS?
Sama sekali tidak. VIGEN adalah engine lokal yang berjalan di ekosistem kamu sendiri. Tidak ada cloud server dari pihak kami, sehingga kamu bebas melakukan render video sepuasnya secara gratis (hanya mengkonsumsi daya HP/Server kamu).
Absolutely not. VIGEN is a local engine running seamlessly within your device sandbox securely. There's no ongoing server cost associated, which means unlimited autonomous video processing natively.

HP saya spesifikasinya rendah, apa sanggup?

My device specs are rather low. Will it support it?
Engine berbasis Python di terminal (Termux) jauh lebih ringan dibandingkan kamu membuka aplikasi editor video berat. Proses penyatuan frame dan tipografi sangat teroptimasi menggunakan library CLI. RAM Android minimal sekalipun cukup mumpuni merekayasa puluhan klip per malam.
Because the Python-driven CLI engine runs purely natively without any heavyweight GUI layout timeline, it relies on exceptionally lightweight encoding workflows to merge frames, making a simple Android device adequately able to render dozens of short-form videos overnight.

Tersedia untuk layout dan desain apa saja?

What layout frameworks or designs are bundled?
Hasil video didesain untuk video vertikal 9:16 cocok untuk (TikTok, YouTube Shorts, dan IG Reels). Saat ini tersedia 26 koleksi template dinamis bawaan seperti Berita, Dialog Chat, Motivasi Reminder, QnA Islami, Fakta Aneh, Poll Perbandingan dsb.
Outputs are tuned natively for TikTok, Vertical Shorts, and Reels formats (9:16 aspect ratio). The engine inherently carries 26 structural templates such as Chat Dialogues, Motivation Quotes, Islamic Verses displays, Random Fact Boxes, Comparison Polls and many more.
