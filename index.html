Kenapa VIGEN?

Bukan aplikasi cloud berbayar bulanan. Vigen adalah core engine komersial milik Anda sendiri yang bekerja 24/7 di balik layar layar.

Bulk Automations

Sistem pipeline yang memproses puluhan konten otomatis dari fetching materi, rendering batch, hingga auto-publishing tanpa campur tangan manual.

26 Dinamis Format

Lebih dari sekadar text-on-video. Menyediakan struktur untuk konten psikologi, berita fakta, edukasi, interaksi polls, hingga konten Islami.

Custom AI Generative

Integrasi Gemini API memastikan skrip konten harian tertulis sempurna sesuai target niche Anda lalu disusun langsung menjadi JSON format yang valid.

4 Steps Pipeline

Dari ide hingga upload dikerjakan otomatis dalam arsitektur script berantai.

1

BG Fetcher

bg_fetcher.py

Script otomatis mencari, mendownload, & kompresi konten visual royalty-free beresolusi HD berdasarkan niche referensi.

python3 bg_fetcher.py --niche="nature"
2

AI Content Generation

ai_generate.py

Mengkoneksikan input topik ke AI & memparsing ouput menjadi JSON siap render.

python3 ai_generate.py "mental health" 10
3

Batch FFmpeg Engine

render.py

Sistem inti memproses layout, merender teks kartu dengan Pillow, lalu menyatukannya dengan Video BG dan BGM menggunakan FFmpeg.

python3 render.py --all
4

Auto Multi-platform

upload.py

Hasil MP4 akhir langsung disuntik metadata dan tersebar via API ke multi social channel.

python3 upload.py --platform=all
